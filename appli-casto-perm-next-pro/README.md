# Consignes Permanentes Castorama Claye-Souilly

Application interne React/Vite prête pour Vercel.

## Déploiement

1. Déposer le contenu de ce dossier sur GitHub.
2. Vercel détecte automatiquement Vite.
3. Cliquer sur Deploy.

## Supabase

Le projet est prêt à recevoir Supabase via les variables :

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

Tant que Supabase n'est pas configuré, l'application fonctionne en mode local navigateur.

## Important

Les identifiants de démonstration ne sont plus affichés sur la page de connexion.
